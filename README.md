# Superstore Data Analysis Dashboard

## Overview
Analyzed retail dataset using Excel to identify key business insights.

## Key Insights
- Technology is the most profitable category
- Tables sub-category is causing losses
- Higher discounts reduce profitability
- Canada region has low performance

## Tools Used
- Microsoft Excel (Pivot Tables, Charts, KPI Dashboard)
